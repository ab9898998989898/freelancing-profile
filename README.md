## ğŸ¨ Customization

### Updating Colors

Edit `tailwind.config.js`:

```javascript
theme: {
  extend: {
    colors: {
      primary: {
        DEFAULT: '#0070f3',  // Change main color
        // Add color shades 50-900
      },
      secondary: {
        DEFAULT: '#10b981',  // Change secondary
      },
    },
  },
}
```

### Updating Fonts

1. Add Google Font link to `index.html`:

```html
<link href="https://fonts.googleapis.com/css2?family=YourFont:wght@400;700&display=swap" rel="stylesheet">
```

2. Update `tailwind.config.js`:

```javascript
fontFamily: {
  sans: ['YourFont', 'sans-serif'],
  heading: ['YourHeadingFont', 'sans-serif'],
}
```

### Updating Content

All content is located in the page components:

- **Personal Info**: Edit `src/pages/Home.jsx`
- **Services**: Edit `src/pages/Services.jsx`
- **Projects**: Edit `src/pages/Portfolio.jsx`
- **Courses**: Edit `src/pages/Courses.jsx`

### Updating Google Form Links

Replace placeholder URLs throughout:

```javascript
// In Services.jsx
href="https://forms.google.com/YOUR-ACTUAL-FORM-ID"

// In Courses.jsx
href="https://forms.google.com/YOUR-COURSE-FORM-ID"

// In Portfolio.jsx
href="https://forms.google.com/YOUR-CONTACT-FORM-ID"
```

### Adding New Pages

1. Create component in `src/pages/`:

```jsx
// src/pages/NewPage.jsx
import { motion } from 'framer-motion';

const NewPage = () => {
  return (
    <motion.div
      initial={{ opacity: 0 }}
      animate={{ opacity: 1 }}
      exit={{ opacity: 0 }}
    >
      {/* Your content */}
    </motion.div>
  );
};

export default NewPage;
```

2. Add route in `App.jsx`:

```jsx
import NewPage from './pages/NewPage';

// In Routes
<Route path="/new-page" element={<NewPage />} />
```

3. Add link in `Header.jsx`:

```jsx
{ name: 'New Page', path: '/new-page' }
```
